### C. Pōja Verdon

Aenar has an array a consisting of n integers.

* [1663/A](https://codeforces.com/contest/1663/problem/C)

```cpp
#include <bits/stdc++.h>
using namespace std;
 
int main(){
 
int n, ele, sum = 0;
cin >> n;
 
for(int i=0; i<n; i++){
    cin >> ele;
    sum += ele;
}
 
cout << sum << endl;
 
}
```
